# next-proper

Easily compile NextJS props via composed methods.

## Install

Via [npm](https://npmjs.com/package/next-proper)

```
npm install --save next-proper
```

Via [Yarn](https://yarn.fyi/next-proper)

```
yarn add next-proper
```
